# Minecraft Clone

Haqiqiy Minecraftga o'xshash 3D voxel o'yin - Three.js va vanilla JavaScript bilan yaratilgan.

## Xususiyatlar

- 3D voxel dunyo - yaxshilangan terrain generatsiyasi bilan
- Birinchi shaxs kamera boshqaruvi - pastga qarash qo'llab-quvvatlanadi
- Bloklarni buzish va qo'yish - progress bilan
- 8 xil blok turi (grass, dirt, stone, wood, sand, cobblestone, leaves, glass)
- Yaxshilangan fizika va harakat
- AABB collision detection
- Inventar tizimi (8 slot)
- Blok nomini ko'rsatish
- Yaxshilangan dunyo generatsiyasi - daraxtlar, qum maydonlari
- Yaxshilangan yoritish va soyalar

## Qanday O'ynash

1. `index.html` faylini zamonaviy brauzerda oching (Chrome, Firefox, Edge)
2. O'yin oynasiga bosing (sichqoncha qulflanadi)
3. Quyidagi boshqaruvlardan foydalaning:
   - **WASD** - Harakat qilish
   - **Space** - Sakrash
   - **Sichqoncha** - Qarash (yuqoriga va pastga)
   - **Chap Tugma** - Bloklarni buzish (ushlab turish kerak)
   - **O'ng Tugma** - Blok qo'yish
   - **1-8** - Inventardan blok turini tanlash
   - **ESC** - Sichqonchani qulfdan ochish

## Talablar

- JavaScript yoqilgan zamonaviy brauzer
- O'rnatish kerak emas - faqat HTML faylni oching!

## Texnik Tafsilotlar

- Three.js r128 bilan qurilgan
- 3D render uchun WebGL
- Yaxshilangan fizika va collision detection
- Procedural terrain generatsiyasi
- AABB (Axis-Aligned Bounding Box) collision detection
- Raycasting bloklar bilan o'zaro ta'sir uchun
